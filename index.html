Webhook & API

Receive real-time notifications when specific events happen in your dashboard.

No webhooks configured yet. Click Add Webhook to get started.

Connect this dashboard with your automation tools like Pabbly Connect, Zapier, or any custom integration.

API Key
Active

Include in request header: Authorization: Bearer YOUR_API_KEY

Base URL
https://api.dashboard.pabbly.com/v1
Available Endpoints
Method Endpoint Description
POST /{ad_account}/leads Add a new lead entry for the specified ad account
POST /{ad_account}/sales Add a new sales entry for the specified ad account

Note: Replace {ad_account} with your ad account name (e.g., pabbly-connect, pabbly-chatflow, pabbly-subscription). Each ad account has its own settings and data.

Quick Example
// Add a new lead for Pabbly Connect ad account
fetch('https://api.dashboard.pabbly.com/v1/pabbly-connect/leads', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer dbk_demo_a1b2c3...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    "name": "John Doe",
    "email": "john@example.com",
    "status": "New"
  })
})
Sample Response
{
  "status": "success",
  "message": "Lead added successfully",
  "ad_account": "pabbly-connect",
  "data": {
    "id": 124,
    "name": "John Doe",
    "email": "john@example.com",
    "status": "New",
    "created_at": "2026-01-28T10:30:00Z"
  }
}
